Gobblegum Machine

"Chew until you die!" - The slogan on Gobblegum machines

Planning
The Gobblegum machine is a machine that can be interacted with within the game Call of Duty: Black Ops III and dispenses a random gumball or "gobblegum" for the player. When interacted with the machine lights flash and the dome lights up with electricity before dispensing a gumball from the lion's mouth. When a gumball is dispensed the lights change color to match the color of the gumball. With these things in mind I decided to came up with a basic list of functions I wanted this machine to accomplish.
    The machine should...
  1. have "dancing" lights when idle.
  2. be able to accept nickels to start the gumball dispensing process.
  3. successfully dispense a gumball through the lion's mouth on the top of the machine.
  4. have the glass ball flash blue and have the side lights flash when dispensing a gumball.
  5. have the side lights flash and change color to match one of the 4 basic gumball colors (green, blue, purple, orange).
With this in mind there are several mechanisms and electronics that will have to function togther in order for the machine to operate as intended. Each of the lights should have 4 colors to match the gumball colors. There should also be some kind of mechanical mechanism to ensure that only nickels are accepted by the mechanism. In order to actually dispense the gumball a clear tube will need to allow for a vertical transference to the lion's head. Finally, a device should recognize the color of the gumball dispensed and send a signal to the lights to change to that color.
